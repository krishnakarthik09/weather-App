# Weather App 🌦️

A simple Weather Application built using **Python**, **Tkinter**, and the **OpenWeatherMap API**. This application allows users to enter a city name and view real-time weather information.

## Features

* Search weather by city name
* Display temperature in Celsius
* Show "feels like" temperature
* Display weather condition and description
* Show humidity and pressure
* Display wind speed
* Show visibility
* Display latitude and longitude
* User-friendly GUI using Tkinter

## Technologies Used

* Python
* Tkinter
* Requests Library
* OpenWeatherMap API

## Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/weather-app.git
cd weather-app
```

### 2. Install Required Library

```bash
pip install requests
```

### 3. Get OpenWeatherMap API Key

1. Create an account at https://openweathermap.org
2. Generate your API key
3. Replace the API key in the code:

```python
API_KEY = "YOUR_API_KEY"
```

### 4. Run the Application

```bash
python weather_app.py
```
